# Proyecto SIGE - Sistema de Facturación

Sistema de gestión para la facturación y control de inventarios de Peluches Salem

## 🚀 Funcionalidades
- Gestión de remisiones.
- Control de inventario y stock en tiempo real.
- Generación de archivos PDF para remisiones.
- Sistema de búsqueda y filtrado de clientes.

## 🛠️ Tecnologías utilizadas
- Python 3.x
- MySQL (Conector de base de datos)
- ReportLab (Generación de PDFs)

## ⚙️ Instalación
1. Clona este repositorio:
   `git clone https://github.com/tucodesystem/proyecto-sige.git`
2. Crea y activa tu entorno virtual:
   `python -m venv .venv`
   `source .venv/bin/activate` (o `.venv\Scripts\activate` en Windows)
3. Instala las dependencias:
   `pip install -r requirements.txt`
